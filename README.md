# Customer-Lifecycle-Analysis-RFM-Segmentation-CLTV-Estimation-and-Churn-Prediction
Bu proje, e-ticaret müşterilerinin geçmiş davranışlarını analiz ederek RFM segmentasyonu oluşturur, müşteri yaşam boyu değerini (CLTV) tahmin eder ve Random Forest ile churn riskini öngörür; böylece hedefli pazarlama ve müşteri sadakati stratejilerini destekler.
